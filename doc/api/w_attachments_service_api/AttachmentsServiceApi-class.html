AttachmentsServiceApi

is a Headless SDK that allows server interaction for attachments without having to load the full attachments module.

Client

This class requires a bigsky client. This is used for the authenticated requests to the server.

Upload Status

When uploadFile is called, all statuses are piped through the uploadStatusStream, so consumers should be subscribing to that stream in order to receive uploadStatus objects.

_serviceApi.uploadStatusStream.listen(_addUpdateTestAttachment);

To dispose of an AttachmentsServiceApi object:

Disposable is used here, so to destroy any invocations, use dispose() which terminates the StreamController and closes the client, cancelling or closing any outstanding connections.
